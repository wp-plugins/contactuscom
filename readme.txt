Add Contact Form from ContactUs.com account to your WordPress Website

== Description ==

Enable ContactUS.com Form in your website.

= Features =

1. Enable ContactUS.com Form in your website.
2. You can manage the visibility of the ContactUs Tab
3. Copy/Paste the JavaScript code provided by your ContactUs.com account


== Installation ==

1. Download ContactUs.com WordPress plug-in from inside the Client Admin.
2. Log into your WordPress page, and go to the “Plugins” section in the menu.
3. Choose “Add New”, and select the option to upload a plugin in .zip format.
4. You’ll be taken to a page similar to www.yoursite.com/wp-admin/plugin-install.php?tab=upload. Upload the ContactUs.com .zip file here, and then press “Install Now”.
5. Upon successful upload, you should see the following message.  Activate the plugin by clicking the “Activate Plugin” link.
6. Once activated, you will see the ContactUs.com Form Plugin listed under your Plugins module.
7. In the side menu, beneath the standard WordPress menu items, you should now see “ContactUs.com” listed with your other plugins.  Click on the ContactUs.com link.
8. You will be automatically redirected to the newly-created “ContactUs.com Form Plugin for WordPress page” within your WordPress admin.
9. Copy/paste your own ContactUs.com website code into the “ContactUs.com JavaScript Code”.
10. Press “Save Changes”.
11. Go back to your ContactUs.com Client Admin, to the "Website Code" section.  Press “Check Code Installation” to check that the code has been installed corrected.
12. Open up your website to confirm that the ContactUs form experience is online.
